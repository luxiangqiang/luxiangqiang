4、路由

  • 将不同的请求，分配给相应的处理函数
  • 区分：路径、请求方法
4.1 路由实现的三种方法
  • path（Post、Get）
  • Router
  • route
4.3 Router 方式

同一路由下的多个子路由。针对于同一类型请求（post、get）下不同的请求。

4.4 rote

针对不同路由（文件夹）下的不同方法下的请求。

4.5 路由参数

用户通过 URL 传递过来的参数，用作路由参数来进行请求操作。

5、中间件

  • Connect: Node.js 的中间件框架；
  • 分层处理；
  • 每层实现一个功能。

如：next() 操作就是一个中间件，会继续执行下一层的功能。
